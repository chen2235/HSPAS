### 任務說明

1. 我會貼上一份 `skill.md`，裡面定義了一個「台股每日行情日曆查詢系統」，包含：  
   - ASP.NET Core 9 Web API 規格  
   - SQL Server 資料表設計  
   - 前端 HTML5 + Bootstrap 的頁面需求（日曆介面、報表）  
   - 歷史回補工具（管理者用）的 API 與前端頁面規格  

2. 請你根據 `skill.md`：  
   - 設計適合的解決方案與專案結構（資料夾與命名空間）。  
   - 實作資料庫實體／DbContext／Migration（EF Core 或你建議的 ORM）。  
   - 實作 Web API Controller、Service、Repository，包含：  
     - `/api/calendar/available-dates`  
     - `/api/daily-prices/by-date`  
     - `/api/daily-prices/{stockId}/history`  
     - `/api/history/backfill`（歷史回補工具）  
   - 實作前端 HTML + Bootstrap 頁面：  
     - `/calendar`：月曆 + 當日行情列表  
     - `/stock/{stockId}`：個股歷史走勢（可用簡單圖表）  
     - `/admin/history-backfill`：歷史回補管理頁  

3. 請特別注意：  
   - 不要使用排程或定時工作，所有資料抓取只在使用者觸發查詢或歷史回補時進行。  
   - 抓台股資料時需呼叫臺灣證券交易所的公開資料 API（依 `skill.md` 規格），解析 CSV/JSON 後寫入 MSSQL。  
   - 請將與 TWSE 溝通的邏輯抽成獨立 Service / Provider，方便之後更換資料來源。  
   - 所有程式碼請盡量採用乾淨、可維護的結構，並在關鍵類別上加簡短註解說明職責。
